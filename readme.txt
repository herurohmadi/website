## ğŸš€ Cara Menggunakan

1. Ekstrak seluruh file ke dalam folder project Anda.
2. Buka `index.html` di browser untuk melihat tampilan website.
3. Edit file HTML pada folder utama atau di dalam folder `/pages/` sesuai kebutuhan.
4. Untuk mengubah tampilan, edit file CSS di `/assets/css/`.
5. Untuk menambah/mengubah fungsi interaktif, edit file JavaScript di `/assets/js/`.
6. Tambahkan gambar baru ke `/assets/images/` jika diperlukan.
7. Jika ingin menambah halaman baru, duplikat salah satu file HTML lalu sesuaikan isinya.
